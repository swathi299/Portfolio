FITNESS TRACKER APP

An android application is feature rich and thus enables users to track their fitness activities. The app has user registration and profile creation page. The main menu has the options such as Workout, Exercises, Weight Track, Body Track, Settings, About. The display menu has 2 options Body Building and Cardio. The data entered by the user is stored in the database. The user can see the performed exercises in History tab. The activities done by the user can seen in Graph also. The users have the option to play MP3 music as well.

PREDICTIONS OF COVID-19 PATIENT ADMISSION IN NHS

The admission of the patients can be predicted with the help of AI so that potential measures can be designed and developed based on the increase in the number of covid-19 patients. The source of the data is taken from NHS. The data from 12 October 2020 to 22 April 2021 is taken in a separate excel sheet for the use of the algorithms and executing the python code. The algorithms such as Logical Regression, Gaussian Naive Bayes, K Nearest Neighbours, Classification and Regression Trees, Linear Discrimnant Analysis are evaluated to predict the patient admission.
